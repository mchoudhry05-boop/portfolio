This repository contains the source code for my personal portfolio. It is built entirely without frameworks to ensure maximum performance and complete control over the render loop. 

### Technical Stack
* **Architecture:** Vanilla HTML, CSS, and JavaScript.
* **Animations:** Custom HTML5 `<canvas>` rendering and `requestAnimationFrame`.
* **Features:** * Frame-perfect video scrubbing tied to scroll position.
  * Interactive flow-field neural backgrounds.
  * Scroll-driven particle text morphing.
  * 3D tilt-shift mathematics for project cards.

### The Narrative
I am transitioning from a 6-year career in Amazon operations to product management, UI/UX design, and AI-assisted building. This site is the bridge between diagnosing bad UX on the frontline and actually building the fix.
